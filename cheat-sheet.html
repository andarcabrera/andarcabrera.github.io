HTML Cheat Sheet

Block Element

A block-level element always starts on a new line and takes up the full width available

div The div tag defines a division or a section in an HTML document.
h1-h6 The h1 to h6 tags are used to define HTML headings. h1 defines the most important heading. h6 defines the least important heading.
p The p tag defines a paragraph.
ul The ul tag defines an unordered list.
li The li tag defines an item in a list.
table The table tag defines an HTML table. An HTML table consists of the table element and one or more tr, th, and td elements. The tr element defines a table row, the th element defines a table header, and the td element defines a table cell.

Inline Elements

An inline element does not start on a new line and only takes up as much width as necessary

span The span tag is used to group inline-elements in a document. The span tag provides no visual change by itself. The span tag provides a way to add a hook to a part of a text or a part of a document.
strong the strong tag defines important text.
em the em tag defines emphasized text.
img the img tag defines images.
br the br tag produces a line break.
input The input tag specifies an input field where the user can enter data.

Self-Enclosing Elements

Self enclosing elements don't require a closing tag. They are represented by just the opening tab.

br Inserts a single line break
img Inserts an image
meta Provides metadata about the HTML document
embeded Defines a container for an external application or interactive content (a plug-in)
param Define parameters for plugins embedded with an
hr Define a thematic change in the content

Structurally Based Elements

These elements provide structure to the layout and they are the norm with HTML5. Below are the most commonly used structural elements:

header Defines a header for the document or a section
nav Defines navigation links in the document
section Defines a section in the document
aside Defines content aside from the page content
footer Defines a footer for the document or a section
article Defines an article in the document
