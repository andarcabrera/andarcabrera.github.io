Classes are a great way to group objects together based on their shared methods. These methods created within a class that are available to all objects of that class are called instance methods , as opposed to singleton methods that are defined for one particular object.

While the objects of a class share the instance methods, they also come with their own state , which is a set of values that are particular to that object. We want to be able to access or set these values that define the state of the object and that is where the instance variables come into play. Instance variables that are initialized within a class can be used by any method defined in that class. After this brief introduction I will use an example of a class and explain as I go the instance methods, variables, and different steps in defining the class.

For my example I will chose a class Book , that will contain instance variables for title, author, price and methods for a description of the books and file as name.

When I initialize the class I want to set the objects' state (title, author, price). I can do that by passing them as arguments to the object and than associate the instance variable with the corresponding argument: @title = title. I also need a way to read or reset the value for these variables. In this case, the value for title and author will not change so I can use and attribute reader method and I need to be able to both read and update the price, so I will set an attribute accessor method for the price instance variable.

  • class Book def initialize (title, author, price)
      @title = title
      @author = author
      @price = price
    end
    attr_reader :title, :author
    attr_accessor :price

Moving on to methods, I want this class to have a display method, that will put out a string with the book's title, author and price, and a file_as method that will show that the book is filed by author. The extra end there is to close the class definition.

  • def display
    "This is #{@title} by #{@author} and it costs $#{price}"
    end
    def file_as
    @author
    end
    end

Now on testing our class we can see if our methods do what we expect them to and check if we can reset the price variable

  • #initialize an instance of Book class
    book = Book.new("Infine Jest", "David Foster Wallace", 30)
    #calling the methods on the object
    p book.title
    p book.author
    p book.price
    p book.display
    p book.file_as
    #changing the price variable
    book.price = 25
    p book.price
    p book.display
